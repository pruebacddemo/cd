Requisitos Técnicos

Hardware mínimo requerido
PC Intel Celeron 266 mhz / AMD K6 266 mhz - 64 MB de RAM - Placa de Video SVGA 24 bits 800 x 600 - Lectora CD 32x

Software mínimo requerido
Windows 98 / Internet Explorer 5.0 / Microsoft Word

Recomendaciones
Si posee Windows XP con Service Pack 2, tenga la precaución de desbloquear la ejecución de contenido activo haciendo click con el mouse sobre la barra de notificación

Atención
El CD ha sido programado para visualizarse con una resolución de video de 1024 x 768
